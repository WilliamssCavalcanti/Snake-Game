#  Snake Game || Jogo Da Cobrinha

Este é um projeto do famoso **Jogo da Cobrinha**, desenvolvido como parte do desafio `#100DaysOfCode` com Python.  
Foi criado usando a biblioteca `turtle`, abordando conceitos de orientação a objetos, colisões e controle por teclado.
Apesar de "simples" foi um bom desafio para desenvolver minhas habilidades com o python

##  Controles
- `↑` Cima
- `↓` Baixo
- `←` Esquerda
- `→` Direita

##  Funcionalidades
- Movimento contínuo da cobra
- Detecção de colisão com comida, parede e consigo mesma
- Crescimento da cobra ao comer
- Sistema de pontuação

##  Tecnologias
- Python 3.x
- Módulo `turtle`

##  Estrutura
├── main.py
├── snake.py
├── food.py
├── scoreboard.py
└── README.md

## English 
This is a project of the classic **Snake Game**, developed as part of the `#100DaysOfCode` challenge with Python.  
It was built using the `turtle` module, covering concepts like object-oriented programming, collision detection, and keyboard control.
Despite "simple" it was a good challenge to develop my skills with python.

##  Controls
- `↑` Up
- `↓` Down
- `←` Left
- `→` Right

##  Features
- Continuous snake movement
- Collision detection with food, walls, and self
- Snake grows when it eats
- Score system

##  Technologies
- Python 3.x
- `turtle` module
